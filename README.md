# HealthSphere – Digital Healthcare Platform

HealthSphere is a Flask-based healthcare web application designed to simplify hospital management and improve patient experience. It provides essential features like appointment booking, patient registration, and secure login in a clean and user-friendly interface.

---

##  Features

-  Appointment Booking System  
-  Patient Registration & Management  
-  Secure Login System  
-  Hospital Services Information  
-  Responsive & Clean UI  

---

##  Tech Stack

- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS  
- **Database:** SQLite / Text Files
